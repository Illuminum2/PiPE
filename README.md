# PiCam MPE

The Raspberry Pi Zero 2W is configured as a USB ethernet gadget.

- Pi ethernet address: `10.55.0.1`
- Laptop ethernet address: `10.55.0.2`
- Stream port: `5555`

Run the Pi server:

```
~/picam_stream/run
```

Run the laptop viewer:

```
./laptop/run
```

Press `q` in the window to exit the viewer.
